# zafu

## Installation
`npm install -g zafu`

## Usage
```

  Usage: zafu [options] <minutes>

  Simple command line meditation timer with a nice bell sound.

  Options:

    -h, --help    output usage information
    -s, --silent  Silence the bell sound before and after the meditation

```
